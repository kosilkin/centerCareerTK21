Вход

Панель управления сайтом

Изменения доступны только пользователю с ролью admin и сохраняются в таблицу site_content, строка id = 1. Слева выберите раздел, справа измените данные и сохраните результат.

Разделы редактирования

Интерфейс разделён на отдельные области, чтобы не держать всё в одном длинном окне.

Тексты сайта

Здесь изменяются заголовки и текстовые блоки на главной странице. Сначала можно применить тексты на странице, затем сохранить их в базу.

Основные тексты

Подсказка

Используйте этот раздел для редактирования структуры текстов без изменения остальных данных сайта. После изменения нажмите «Применить тексты на странице», чтобы сразу увидеть результат, и затем сохраните всё в базу.

Запись в центр

Здесь создаются и редактируются карточки записи. Для каждой записи можно указать описание, лимит мест, даты и времени, а также пояснение для студентов.

Форма редактирования

Список существующих записей

Справа отображаются уже созданные карточки. Выберите нужную запись для редактирования или удаления.

Мероприятия

Здесь редактируются события календаря. Можно задать дату, время, место, цвет отображения, лимит мест и отдельные слоты записи.

Форма редактирования

Список событий

Выберите событие из списка, чтобы исправить дату, время, описание или параметры записи.

Новости

Здесь можно публиковать новости центра карьеры. Для новости доступны дата, краткое описание, полный текст, обложка и ссылка на видеоролик.

Форма редактирования

Список новостей

На главной странице показываются только три последние опубликованные новости. На странице новостей выводится весь опубликованный архив.

Сотрудники

Здесь редактируется блок команды. Можно добавить должность, описание и фотографию по ссылке или загрузкой файла.

Форма редактирования

Можно вставить ссылку или выбрать файл. Загруженный файл сохраняется прямо в содержимом сайта.

Список сотрудников

В списке справа можно открыть сотрудника на редактирование и быстро проверить, как он будет выглядеть в блоке команды.

Стажировки

Здесь редактируются карточки стажировок. Блок размещается на главной странице над направлениями.

Форма редактирования

Список стажировок

Используйте этот раздел для публикации практик и стажировок отдельно от консультаций и мероприятий.

Направления

Здесь редактируются карточки направлений и отдельные страницы по каждому направлению с описанием, материалами и кнопками перехода.

Форма редактирования

Список направлений

Список справа помогает быстро переключаться между карточками и не прокручивать длинное окно вверх и вниз.
